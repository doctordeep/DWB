Why a standalone Twitter widget?

Since this piece was written Twitter has discontinued their API Version 1.0 and introduced Version 1.1. This means that the method to create an embedded Twitter widget has also changed. The following has been updated to reflect the changes brought about the new Twitter API.

Many versions of the Twitter Search Widget have been customised for CMSs such as Wordpress. Following some simple instructions, these are quite easy to install. For non CMS websites, there are a number of bespoke Twitter search clients available, many of which offer powerful options.

Locally hosting the standard Twitter Search Widget has the advantage of simplicity. There is the attraction that as the Twitter Widget is purely JavaScript, it can be also be inserted into static HTML pages. Two examples of the Twitter Widget are demonstrated, a search widget and a list widget.

Instructions for use

  • Download and unzip the twitter_search_widget.zip file to a local disc. The twitter_search_widget.zip file is a replica of this example and can run from the local disc or a USB stick.
  • Alternatively, copy the file and directories to your website. If the directories already exist just copy the files into them
  • To replace the current examples with your own, navigate to the Widget Configurator on your twitter account. You can create widgets to display your own tweets, lists etc.
  • The generated HTML and Javascript code looks like:
                               Tweets by @donboulton
    
                           
             National Park Tweets
  • For pages you wish a Twitter Widget to appear simply paste the Twitter Widget code(s) in the HTML body.

Changing the appearance ot the Twitter widget

The appearance of the widget can be customised by following the examples in Twitter's embedded timelines guide.

Limitations and bugs

  • Only publicly available accounts and lists can be displayed for which an ID is known
  • Some users that tweet excessively or engage in too much # may not appear on the Twitter feeds. This will give a blank result in the Twitter Widget window
